# Public Assets

## Notification Icons

Los archivos `icon-192.png` y `badge-72.png` actualmente son placeholders SVG.

**Para producción, debes reemplazarlos con imágenes PNG reales:**

- **icon-192.png**: Ícono principal de 192x192 píxeles (formato PNG)
- **badge-72.png**: Badge pequeño de 72x72 píxeles (formato PNG)

Puedes crear estos íconos usando herramientas como:
- [Figma](https://figma.com)
- [Canva](https://canva.com)
- [RealFaviconGenerator](https://realfavicongenerator.net/)

El ícono debe representar la aplicación de finanzas personales (por ejemplo, un símbolo de dólar, billetera, o gráfico).
